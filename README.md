## ✨ 核心功能 (Features)

### 1. 🖱️ 聊天栏快捷操作 (+1 & 复制)
* **无缝交互**：鼠标悬停在右侧聊天列表的任意弹幕上，自动浮现 `[+1]` 和 `[复制]` 按钮。
* **一键复读**：看到神级烂梗？点击 `+1` 瞬间自动发送，无需手动复制粘贴。
* **白嫖好梗**：点击 `复制` 一键复制到系统剪贴板，按钮自带“成功!”视觉反馈。

### 2. 🦼 赛博独轮车 (内置弹幕草稿箱)
* **双模循环**：
  * **单梗循环**：对着一句话死磕。
  * **列表循环**：按顺序轮播你存入的烂梗。
* **安全防风控**：可自定义发送间隔（默认 6 秒，建议不低于此值以防被系统吞弹幕）。
* **持久化存储**：烂梗存在本地，不用担心丢失。
* **视觉跑马灯**：开启循环后，列表内高亮指示当前正在发送的弹幕，并支持随时一键刹车。

### 3. 🛡️ 智能防刷屏净化 (Anti-Spam)
* **护眼模式**：直播间烂梗太多看不过来？一键开启防刷屏。
* **底层折叠**：自动监听底层DOM变动，遇到**10 秒内**出现的完全重复的弹幕，直接在本地隐藏，不影响网页性能，还你清爽的观播体验。

### 4. 🎯 播放器原生右键菜单融合 (终极杀器)
* **利用官方接口**：直接在直播画面中**右键点击**飘过的弹幕，在B站官方弹出的菜单中，注入蓝色的 `🚀 +1 (发送)` 选项。直播的官方菜单有复制弹幕功能，这里只实现了+1，复制可以用官方的。
* **原生体验**：完美继承 B 站新版 Webpack 混淆的 UI 样式和悬浮动画；点击发送后，脚本会模拟人类操作执行官方的“关闭”操作，不留痕迹，防止框架状态卡死。

## 🛠️ 安装指南 (Installation)

1. **安装浏览器插件管理器**：
   * 推荐安装 [Tampermonkey (油猴)](https://www.tampermonkey.net/) 插件（支持 Chrome / Edge / Firefox / Safari）。
2. **安装本脚本**：
   * 在 Tampermonkey 中点击 **“添加新脚本”**。
   * 将本仓库中的脚本源码（`bili-danmaku-assistant.js`）全选复制并粘贴进去。
   * 点击 **文件 -> 保存** (或 `Ctrl+S`)。
3. **尽情享用**：
   * 打开任意一个 [B站直播间](https://live.bilibili.com/)，即可爽喷
